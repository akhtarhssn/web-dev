HTML Introduction

HTML stands for HyperText Markup Language. It is the standard markup language for creating web pages and web applications.

What is HTML?

HTML is used to structure content on the web. It consists of a series of elements that you use to enclose, or wrap, different parts of the content to make it appear or act in a certain way.

Example

                    
<!DOCTYPE html>
<html>
    <head>
        <title>My First Web Page</title>
    </head>
    <body>
        <h1>Welcome to HTML</h1>
        <p>This is a paragraph.</p>
    </body>
</html>

The above code creates a simple web page with a heading and a paragraph.
